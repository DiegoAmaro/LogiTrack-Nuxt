# LogiTrack-Nuxt 🚚
> **Business Digitalization Solution for Transport Logistics**

## 📌 Overview (Visão Geral)
LogiTrack is a specialized management tool designed to streamline freight control and logistics operations for small to medium-sized transport companies. 

*Português: O LogiTrack é uma ferramenta de gestão especializada, projetada para otimizar o controle de fretes e operações logísticas para pequenas e médias empresas de transporte.*

## 🚀 The Challenge (O Desafio)
Managing freight costs, vehicle maintenance, and driver schedules manually or through fragmented spreadsheets often leads to data loss and financial inefficiency. This project aims to centralize these operations into a high-performance web interface.

*Português: Gerenciar custos de frete, manutenção de veículos e escalas de motoristas manualmente ou por planilhas fragmentadas gera perda de dados e ineficiência. Este projeto centraliza essas operações.*

## 🛠️ Tech Stack (Tecnologias)
- **Framework:** Nuxt 3 (Vue.js)
- **Language:** TypeScript (Type safety and scalability)
- **Styling:** Tailwind CSS
- **Backend/Database:** Supabase (PostgreSQL)
- **State Management:** Pinia

## 🏗️ Software Engineering Principles (Princípios de Engenharia)
- **Clean Code:** Focused on readability and maintainability.
- **Component-Based Architecture:** Modular UI components for reuse.
- **Responsive Design:** Mobile-first approach for drivers on the road.
- **Documentation:** Full requirements analysis and API documentation.
